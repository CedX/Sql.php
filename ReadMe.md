# SQL Mapper for PHP
A simple micro-ORM for [PHP](https://www.php.net), based on [PDO](https://www.php.net/manual/en/book.pdo.php) and data annotations.  
If you like [SQL](https://en.wikipedia.org/wiki/SQL) but dislike the boilerplate of PDO: **SQL Mapper for PHP** might be for you!

> Think of [Dapper](https://www.learndapper.com) and its alternatives, but less advanced/powerful.
